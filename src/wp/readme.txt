Online-консультант Me-Talk для вашего сайта.

== Description ==

Данный плагин позволит вам легко вставить код Online-консультанта Me-Talk на ваш сайт. При этом вам не нужно редактировать файлы вашей темы, и счётчик продолжит работать даже при смене вашей активной темы.

После активации плагина зайдите в раздел Настройки - Me-Talk для того, чтобы вставить код вашего виджета. Учтите, что для редактирования кода виджета, вам необходимы права администратора на сайте WordPress (или супер-администратора в режиме Multisite).

== Installation ==

1. Распакуйте архив с плагином
2. Загрузите папку с плагином в директорию `/wp-content/plugins/` на вашем сервере
3. Активируйте плагин Me-Talk в разделе Плагины в панели администрирования WordPress
4. Зайдите в раздел Настройки - Me-Talk для вставки кода вашего виджета
